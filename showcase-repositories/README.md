# Showcase Repositories

Este directorio agrupa cuatro repositorios locales listos para evolucionar a proyectos publicables del portafolio.

Objetivo:
- convertir cada tarjeta del portafolio en una pieza técnica defendible;
- documentar arquitectura, alcance, riesgos, y cumplimiento mínimo esperado;
- facilitar una futura publicación en GitHub con narrativa profesional para LinkedIn y entrevistas.

Repositorios incluidos:
- `infra-vps-hardening-lab`
- `secure-cicd-platform`
- `blue-team-observability-lab`
- `network-segmentation-lab`

Estado actual:
- base documental y de arquitectura creada;
- roadmaps definidos por fases;
- lineamientos de cumplimiento y marco legal identificados;
- pendientes de implementación completa repo por repo.

Nota importante:
Estas bases no sustituyen asesoría jurídica, auditoría ISO formal ni autorización operativa para pruebas reales. Sirven como repositorio de diseño, planificación y futura ejecución controlada.
